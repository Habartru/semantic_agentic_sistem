# Промт для Claude Design — Лендинг SEO Agents

## Задача

Создай современный, профессиональный лендинг для SaaS-продукта "SEO Agents" — AI-платформы для автоматизации работы с коммерческим семантическим ядром. Лендинг должен объяснять продукт простым языком, показывать технологичность и вызывать доверие. Целевая аудитория — SEO-специалисты, маркетологи, владельцы интернет-магазинов и агентства.

Язык лендинга — русский. Стиль — современный tech/AI SaaS (вдохновение: Linear, Vercel, Notion AI, Jasper). Тёмная тема с яркими акцентами — ощущение AI-продукта.

---

## Цветовая схема лендинга

- **Фон:** Тёмный градиент — от #0F0F23 (тёмно-синий/чёрный) к #1A1A2E
- **Акцент Primary:** Индиго (#6366F1) → пурпурный (#8B5CF6) — градиент для кнопок и акцентов
- **Акцент Secondary:** Голубой (#38BDF8) — для подсветки данных и графиков
- **Success:** Зелёный (#34D399) — для метрик и позитивных чисел
- **Текст основной:** Белый (#F8FAFC)
- **Текст вторичный:** Серо-голубой (#94A3B8)
- **Карточки:** Полупрозрачный (#1E293B с opacity) + border (#334155)
- **Glow-эффекты:** Индиго/пурпурные свечения за элементами

## Секции лендинга

### 1. Hero-секция

**Заголовок (H1):**
"AI-агенты для семантического ядра — от сбора до готового плана действий"

**Подзаголовок:**
"8 специализированных агентов автоматически собирают, расширяют, кластеризуют и приоритизируют поисковые запросы. Вы получаете не список ключей, а готовый план: какие страницы создать, какие обновить, и в каком порядке."

**CTA-кнопка:** "Попробовать бесплатно" (градиент индиго→пурпурный, большая, с glow)
**Вторая кнопка:** "Смотреть демо" (ghost, с иконкой play)

**Визуал справа/снизу:**
Анимированная схема пайплайна — 8 агентов в цепочке с данными, которые "текут" между ними. Каждый агент — иконка в светящемся круге. Линии-коннекторы с анимацией движения данных. Стиль: как схема нейронной сети или data pipeline.

### 2. Секция "Проблема и решение"

**Заголовок:** "Семантическое ядро — это не список ключей"

Два столбца:

**Левый столбец — "Как обычно"** (приглушённые тона, перечёркнутый стиль):
- Копируете ключи из KeyCollector → получаете 10 000 фраз
- Вручную чистите от мусора → тратите 2 дня
- Группируете в Excel → теряете логику
- Не знаете, что делать с результатом
- Итог: таблица, которая пылится

**Правый столбец — "С SEO Agents"** (яркие тона, галочки):
- AI собирает запросы из 4+ источников автоматически
- Чистит мусор и определяет интент каждого запроса
- Кластеризует с учётом SERP и бизнес-логики
- Маппит на страницы: создать / обновить / не брать
- Итог: план действий с приоритетами

### 3. Секция "Как это работает" — Пайплайн агентов

**Заголовок:** "8 агентов. Один пайплайн. Полная автоматизация."

Горизонтальный таймлайн с 8 шагами. Каждый шаг — карточка с:
- Номер (в светящемся круге)
- Иконка агента
- Название
- Краткое описание (1-2 строки)

**Агенты:**

1. **Research Agent** — "Собирает данные: сайт, конкуренты, Search Console, поисковые подсказки"
   Иконка: лупа

2. **Expansion Agent** — "Расширяет ядро: Google Suggest, Trends, LLM-генерация long-tail запросов"
   Иконка: стрелки расходящиеся

3. **Cleaning Agent** — "Удаляет мусор: дубли, нерелевантные, слишком общие, юридически рискованные"
   Иконка: фильтр/воронка

4. **Intent Agent** — "Определяет интент: коммерческий, информационный, транзакционный, локальный"
   Иконка: мишень

5. **Clustering Agent** — "Группирует запросы: 1 кластер = 1 интент = 1 страница"
   Иконка: группа/кластер

6. **Mapping Agent** — "Маппит на страницы: создать новую, обновить существующую, не брать"
   Иконка: карта/сетка

7. **Prioritization Agent** — "Расставляет приоритеты по бизнес-ценности, конкуренции и потенциалу"
   Иконка: график растущий

8. **Feedback Agent** — "Анализирует результаты после публикации и предлагает улучшения"
   Иконка: обновление/цикл

При наведении на шаг — карточка раскрывается с дополнительным описанием.

### 5. Секция "Преимущества"

**Заголовок:** "Почему SEO Agents"

Сетка 2×3 карточки:

1. **Полный цикл** — "От seed-ключей до плана действий за один запуск. Не нужно переключаться между инструментами."

2. **AI-классификация** — "LLM определяет интент, оценивает бизнес-ценность и предлагает тип страницы для каждого запроса."

3. **Умная кластеризация** — "Не по похожести слов, а по SERP-логике. Семантически близкие фразы могут требовать разных страниц."

4. **Реальные источники** — "Google Suggest, Trends, конкуренты, Search Console — не галлюцинации AI, а реальные данные."

5. **Приоритизация** — "Scoring-модель из 7 факторов: спрос, бизнес-ценность, конкуренция, интент, тренды, контент-гэп, каннибализация."

6. **Экспорт и интеграция** — "Результаты в CSV, таблицы с фильтрами, API для интеграции с вашими инструментами."

## Анимации и эффекты

- **Hero:** Плавное появление элементов снизу (fade-in + slide-up)
- **Пайплайн:** Данные "текут" по линиям между агентами (CSS-анимация)
- **Карточки:** Появляются при скролле (Intersection Observer)
- **Кнопки:** Hover — glow-эффект, scale 1.02
- **Фон:** Subtle gradient mesh или dots pattern
- **Числа:** Count-up анимация при скролле в видимую область

## Адаптивность

- Desktop: полная ширина, горизонтальный пайплайн
- Tablet: 2 колонки преимуществ, компактный пайплайн
- Mobile: 1 колонка, вертикальный пайплайн, стек карточек

## Важно

- Лендинг должен ощущаться как продукт уровня Linear/Vercel — технологичный, но понятный
- Тёмная тема подчёркивает AI/tech характер продукта
- Все тексты на русском
- Без стоковых фото — только UI-мокапы, иконки, схемы и абстрактная графика
- Лендинг — это ОТДЕЛЬНАЯ страница, не часть основного приложения (может быть отдельным HTML или route /landing)
