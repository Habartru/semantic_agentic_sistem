# Промт для Claude Design — Дизайн-система SEO Agents

## Описание проекта

SEO Agents — это веб-платформа для автоматизации работы с коммерческим семантическим ядром. Система использует 8 AI-агентов, которые последовательно обрабатывают поисковые запросы: собирают данные, расширяют семантику, чистят мусор, определяют интент, кластеризуют, маппят на страницы, приоритизируют и анализируют результаты.

Целевая аудитория — SEO-специалисты, маркетологи, владельцы интернет-магазинов. Язык интерфейса — русский.

## Задача

Создай полную дизайн-систему и UI Kit для платформы SEO Agents. Нужен профессиональный, чистый, современный дизайн для SaaS-продукта в нише SEO-инструментов. Стиль — минималистичный, data-driven, с фокусом на читаемость таблиц и данных.

## Компоненты UI Kit

### 1. Навигация (Navbar)

- Фиксированная сверху, белый фон, нижний border
- Логотип "SEO Agents" слева (индиго, bold)
- Ссылки справа: "Проекты", "Настройки"
- Активная ссылка — индиго, неактивная — серая
- Высота: 64px

### 2. Кнопки

- **Primary:** Индиго фон (#4F46E5), белый текст, hover: #4338CA
- **Secondary:** Белый фон, индиго border, индиго текст
- **Success:** Зелёный фон (#059669), белый текст (для "Запустить анализ")
- **Danger:** Красный фон, белый текст
- **Ghost:** Прозрачный фон, серый текст, hover: серый фон
- Размеры: sm (32px height), md (40px), lg (48px)
- Состояния: default, hover, active, disabled, loading (с спиннером)

### 3. Карточки (Cards)

- Белый фон, border серый (#E5E7EB), скругление 12px, тень
- Padding: 24px
- Заголовок карточки: 16px semibold
- Используются для: проекты, статистика, настройки

### 4. Таблицы

- Заголовок: серый фон (#F9FAFB), bold, uppercase 12px
- Строки: белый фон, hover серый (#F9FAFB)
- Border между строками: #E5E7EB
- Строки раскрашиваются по приоритету: critical=красный фон, high=зелёный, medium=жёлтый, low=серый
- Сортируемые колонки — иконка стрелки

### 5. Формы

- **Input:** Border серый, скругление 8px, focus: индиго ring
- **Textarea:** То же, высота авто
- **Select/Dropdown:** Как input + стрелка
- **Label:** 14px, medium, серый (#374151), margin-bottom 4px
- **Placeholder:** 14px, светло-серый (#9CA3AF)

### 6. Бейджи (Badges)

- **Critical:** красный фон (#FEE2E2), красный текст (#991B1B)
- **High:** зелёный фон (#D1FAE5), зелёный текст (#065F46)
- **Medium:** жёлтый фон (#FEF3C7), жёлтый текст (#92400E)
- **Low:** серый фон (#F3F4F6), серый текст (#374151)
- Интенты: commercial (синий), transactional (фиолетовый), informational (голубой), local (оранжевый), comparison (розовый)
- Действия: create (зелёный), update (жёлтый), skip (серый), merge (синий), faq (фиолетовый)

### 7. Прогресс-бар пайплайна

Самый важный компонент! Показывает 8 шагов-агентов:

1. Исследование (ResearchAgent)
2. Расширение (ExpansionAgent)
3. Очистка (CleaningAgent)
4. Интент (IntentAgent)
5. Кластеризация (ClusteringAgent)
6. Маппинг (MappingAgent)
7. Приоритизация (PrioritizationAgent)
8. Обратная связь (FeedbackAgent)

Состояния каждого шага:
- **Ожидает:** серый круг с номером, серая линия
- **В процессе:** индиго круг с пульсирующей анимацией, индиго линия (заполняется)
- **Завершён:** зелёный круг с галочкой, зелёная линия
- **Ошибка:** красный круг с крестиком

Горизонтальное расположение на десктопе (8 шагов в ряд с линиями-коннекторами).
Вертикальное на мобильном (список с линией слева).

### 8. Модальные окна

- Оверлей: чёрный 50% прозрачности
- Белый контейнер, скругление 16px, тень
- Заголовок, контент, кнопки внизу
- Кнопка закрытия (X) справа сверху

### 9. Уведомления / Alerts

- Success: зелёный фон, иконка галочки
- Error: красный фон, иконка ошибки
- Warning: жёлтый фон, иконка предупреждения
- Info: синий фон, иконка информации

### 10. Статистические карточки

Маленькие карточки с метриками:
- Число (большое, bold, цветное)
- Подпись (маленькая, серая, uppercase)
- Используются на странице результатов: "Всего ключей", "Кластеров", "Critical", "High", "Medium", "Low"

## Страницы

### Страница 1: Список проектов (/projects)

- Заголовок "Проекты" + кнопка "Новый проект" (primary, справа)
- Сетка карточек проектов (3 колонки на десктопе)
- Каждая карточка: название, URL сайта, дата, кнопка "Открыть"
- Пустое состояние: иллюстрация + текст "Создайте первый проект" + кнопка

### Страница 2: Создание проекта (/projects/new)

- Заголовок "Новый проект"
- Форма в карточке:
  - Название проекта
  - URL сайта
  - Описание бизнеса (textarea)
  - Стартовые ключевые слова (textarea, по одному на строку)
  - Гео (input, default "Москва")
  - URL конкурентов (textarea, по одному на строку)
- Кнопки: "Создать" (primary) + "Отмена" (ghost)

### Страница 3: Проект (/projects/{id})

Три секции:

**Секция 1 — Информация:**
- Карточка с данными проекта (URL, описание, ключи, гео, конкуренты)
- Большая зелёная кнопка "Запустить анализ"

**Секция 2 — Прогресс пайплайна:**
- Прогресс-бар с 8 шагами (компонент описан выше)
- Появляется после нажатия "Запустить анализ"
- Реалтайм-обновление через SSE

**Секция 3 — История запусков:**
- Таблица: дата, статус (бейджи), прогресс, кнопка "Результаты"

### Страница 4: Результаты (/projects/{id}/results/{run_id})

Самая важная страница!

**Блок статистики:**
- 6 карточек в ряд: Всего ключей, Кластеров, Critical (красный), High (зелёный), Medium (жёлтый), Low (серый)

**Блок фильтров:**
- 3 dropdown в ряд: фильтр по интенту, по действию, по приоритету
- Кнопка "Скачать CSV" (secondary, справа)

**Таблица результатов:**
- Колонки: Ключевое слово, Кластер, Интент (бейдж), Страница (URL), Действие (бейдж), Приоритет (бейдж + score), Причина
- Строки группируются визуально по кластерам (разделитель или цвет)
- Строки подсвечиваются по приоритету
- Кликабельные заголовки для сортировки

### Страница 5: Настройки (/settings)

- Секция "API ключи": поле пароля с кнопкой показать/скрыть + индикатор статуса (зелёный/красный) + кнопка "Проверить подключение"
- Секция "Модель LLM": dropdown с моделями
- Секция "Сервисы": таблица статусов (сервис / требует ключ / статус)
- Кнопка "Сохранить" (primary)

## Адаптивность

- **Desktop (1280px+):** полная ширина, 3 колонки карточек, горизонтальный прогресс-бар
- **Tablet (768-1279px):** 2 колонки, компактные таблицы
- **Mobile (< 768px):** 1 колонка, вертикальный прогресс-бар, горизонтальный скролл таблиц

- Frontend: HTML/CSS/JS, Jinja2 шаблоны
- CSS: Tailwind CSS через CDN

- Иконки: Heroicons

## Примечания

- Дизайн должен ощущаться как профессиональный SaaS-инструмент уровня Ahrefs/Semrush, но проще и чище
- Основной фокус — на странице результатов с таблицей, она самая важная для пользователя
- Прогресс-бар пайплайна — ключевая "вау" фича, должен выглядеть впечатляюще
- Все тексты и подписи — на русском языке
